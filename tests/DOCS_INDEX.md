# 測試文檔索引

本目錄包含所有測試相關的文檔。以下是各文檔的說明：

## 📋 測試計劃和說明

### [TEST_PLAN.md](./TEST_PLAN.md)
**測試計劃文件** - 完整的測試計劃，詳細列出所有需要測試的項目，按類別組織：
- 單元測試計劃
- API 端點測試計劃
- 業務邏輯測試計劃
- 整合測試計劃
- 錯誤處理測試計劃
- 性能測試計劃
- 安全性測試計劃
- 前端測試計劃

### [TESTING.md](./TESTING.md)
**測試程序說明** - 測試程序的總體說明：
- 已完成的測試項目
- 測試統計
- 如何執行測試
- 測試覆蓋率目標
- 相關文件連結

## 🔧 測試修復和問題

## 文檔使用建議

1. **開始測試前**：閱讀 [TEST_PLAN.md](./TEST_PLAN.md) 了解測試計劃
2. **執行測試時**：參考 [README.md](./README.md) 了解如何執行測試
3. **查看中文輸出**：閱讀 [TEST_OUTPUT_CN.md](./TEST_OUTPUT_CN.md)
4. **遇到問題時**：查看 [TEST_FIXES.md](./TEST_FIXES.md) 了解常見問題
5. **了解結構**：查看 [TEST_STRUCTURE.md](./TEST_STRUCTURE.md) 了解文件組織

## 相關文件

- **pytest.ini**: pytest 配置文件（位於專案根目錄）
- **run_tests.sh**: 測試執行腳本（位於專案根目錄）
- **run_tests_cn.py**: 中文測試報告腳本（位於專案根目錄）
