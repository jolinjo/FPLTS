# 功能按鈕圖標

此資料夾用於存放功能按鈕的圖標圖片。

## 圖片檔案名稱

請將您的圖標圖片命名為以下檔案名稱：

- `inbound.png` - 貨物遷入按鈕圖標
- `outbound.png` - 貨物遷出按鈕圖標
- `trace.png` - 追溯查詢按鈕圖標
- `first.png` - 首站遷出按鈕圖標

## 圖片規格建議

- **格式**：PNG（支援透明背景）或 SVG
- **尺寸**：建議 128x128 像素或更高（系統會自動縮放）
- **背景**：建議使用透明背景
- **風格**：建議使用簡潔的線條圖標或扁平化設計

## 使用方式

1. 將您的圖標圖片放入此資料夾
2. 按照上述檔案名稱命名
3. 重新載入頁面即可看到新的圖標

## 備註

如果圖片檔案不存在，系統會自動使用預設的 emoji 圖標作為備用顯示。
